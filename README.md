# 📚 Minha Rotina Escolar

Projeto HTML simples que documenta a rotina escolar diária, materiais necessários, disciplinas de tecnologia e o horário semanal de aulas.

---

## 📄 Sobre o Projeto

Este arquivo (`index1.html`) é uma página web estática criada em HTML puro, sem uso de CSS externo ou JavaScript. Seu objetivo é organizar e apresentar de forma clara as informações da rotina de um estudante.

---

## 🗂️ Estrutura do Arquivo

```
index1.html
```

O projeto contém apenas um arquivo HTML com as seguintes seções:

### 1. Rotina Diária
Lista ordenada com as atividades do dia a dia:
- Acordar às 05:20
- Se trocar
- Ir para a escola
- Estudar até 13:10
- Voltar para casa
- Almoçar
- Dormir

### 2. Materiais Escolares
Lista com os itens necessários para as aulas:
- Caderno, Caneta, Mochila, Estojo, Lápis, Branquinho

### 3. Disciplinas de Tecnologia
Matérias da área tecnológica cursadas:
- HTML
- CSS
- Programação de Algoritmo
- Banco de Dados
- Programação Web

### 4. Horário Semanal de Aulas
Tabela com os horários de segunda a sexta-feira, das 7:00 às 12:20, incluindo o intervalo das 9:30 às 9:50.
